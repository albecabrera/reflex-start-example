# Mi primer curso de Python-Reflex (aprendiendo con MoureDev) 

## Instalar reflex

git clone <url-to-this-repo>

```sh
cd link_bio
python3 -m venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt
reflex init
reflex run
```
